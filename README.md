# ISO/IEC 42001 Lead Auditor Practice Suite

This package contains:

- `core.html`: 240 core practice questions across Sets 1 to 3
- `scenarios.html`: 12 multi-question case studies with 60 linked questions
- `index.html`: launcher page for both apps

## Update your GitHub Pages repository

1. Extract this ZIP.
2. Upload all extracted files to the root of your existing GitHub repository.
3. Replace existing files when prompted.
4. Keep GitHub Pages configured to deploy from `main` and `/(root)`.
5. Allow several minutes for the site and offline cache to refresh.

The complete suite contains 300 practice questions.
